PCB-ontwikkeling & embedded programming

Een complete machinebesturing op één printplaat. Wij ontwerpen én programmeren PCB's in eigen beheer, en rekenen eerlijk voor u uit wanneer dat de beste keuze is.

// WAT HET IS

Eén printplaat die uw machine aanstuurt

Een PCB (Printed Circuit Board) is een op maat ontwikkelde printplaat die een complete machine kan aansturen. Waar een besturingskast is opgebouwd uit losse componenten, zit bij een PCB alles op één compacte, herhaalbare plaat: goedkoper per stuk, kleiner en sneller te produceren.

De ontwikkeling van een eigen PCB is een investering die zich meestal pas bij grotere oplagen terugbetaalt. Precies daarom is eerlijk advies hier zo belangrijk.

// WAAROM VAN DIS

Ontwerp én programmering uit één hand

Omdat wij beide disciplines in huis hebben, adviseren we onafhankelijk: soms is een eigen PCB de beste keuze, soms een standaardoplossing.

  • Eigen ontwerp: de printplaat wordt exact ontworpen op uw machine en toepassing.
  • Embedded programming: ook de software op de PCB ontwikkelen we volledig zelf.
  • Eerlijke rekensom: wij rekenen uit vanaf welke oplage een eigen PCB rendabel wordt, en adviseren anders een alternatief.
  • Bewezen in de praktijk: voor partners als Big Ass Battery produceren wij PCB's in serie.
  • Eén partner: de PCB werkt gegarandeerd samen met uw panelen, PLC's en software, want die komen uit dezelfde werkplaats.
// VEELGESTELDE VRAGEN

Veelgestelde vragen

Wat is een PCB precies?

Een PCB (Printed Circuit Board, printplaat) is een op maat ontworpen printplaat die een complete machine kan aansturen. Alle besturing zit op één compacte plaat, in plaats van losse componenten in een kast.

Wanneer is een eigen PCB interessant?

PCB-ontwikkeling is meestal pas rendabel bij grotere oplagen: het ontwerp kost eenmalig geld, daarna dalen de kosten per stuk sterk. Omdat wij ontwerp en programmering in eigen beheer hebben, rekenen we eerlijk voor u uit vanaf welk aantal een eigen PCB de beste keuze is.

Verzorgen jullie ook de software op de PCB?

Ja, de embedded programming doen we volledig zelf. Ontwerp, programmering en test komen uit één hand, dus de printplaat werkt gegarandeerd samen met de rest van uw besturing.
